Tag HTML <meta>

Definição

A tag <meta> define metadados sobre um documento HTML. Metadados são dados (informações) sobre dados.
tags <meta> sempre ficam dentro do elemento <head> e normalmente são usadas para especificar conjunto de caracteres, descrição da página, palavras-chave, autor do documento e configurações da janela de visualização.
Os metadados não serão exibidos na página, mas podem ser analisados ​​por máquina.
Os metadados são usados ​​por navegadores (como exibir conteúdo ou recarregar páginas), mecanismos de pesquisa (palavras-chave) e outros serviços da web.
Existe um método para permitir que os web designers assumam o controle da janela de visualização (a área visível do usuário em uma página da web), por meio da tag <meta>.

Atributos Globais

A tag <meta> também suporta atributos globais em HTML .

Exemplo

Descreva os metadados em um documento HTML:

    <head>
        <meta charset="UTF-8">
        <meta name="description" content="Free Web tutorials">
        <meta name="keywords" content="HTML, CSS, JavaScript">
        <meta name="author" content="John Doe">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    </head>
