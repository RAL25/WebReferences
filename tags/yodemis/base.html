Tag HTML <base>

Definição

A tag <base> especifica o URL base e/ou destino para todos os URLs relativos em um documento.
A tag <base> deve ter um atributo href ou target presente, ou ambos.
Só pode haver um único elemento <base> em um documento e ele deve estar dentro do elemento <head>.

Atributos

Atributo Valor Descrição
href URL Especifica o URL base para todos os URLs relativos na página
target
_blank
_parent
_self
_top
 Especifica o destino padrão para todos os hiperlinks e formulários na página

Atributos Globais

A tag <base> também suporta atributos globais em HTML .

Atributos de Evento

A tag <base> não suporta atributos de evento em HTML .

Exemplo

Especifique um URL padrão e um destino padrão para todos os links de uma página:


    <head>
        <base hreaf="https://www.w3schools.com/" target="_blank">  
    <head\>

    <body>
        <img src="images/stickman.gif" width="24" height="39" alt="Stickman">
        <a href="tags/tag_base.asp>HTML base Tag</a> 
    <body\>
