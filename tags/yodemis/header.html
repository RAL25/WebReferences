Tag HTML <header>

Definição

O elemento <header> representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.
Um elemento <header> normalmente contém:

  • um ou mais elementos de título (<h1> - <h6>)
  • logotipo ou ícone
  • informações de autoria

    • Nota: Você pode ter vários elementos <header> em um documento HTML. No entanto, <header> não pode ser colocado dentro de um <footer>, <address> ou outro elemento <header>.

    Exemplo

    Um cabeçalho para um <article>:

            
        <article>
            <header>
            <h1>A heading here</h1>
            <p>Posted by John Doe</p>
            <p>Some additional information here</p>
            </header>
            <p>Lorem Ipsum dolor set amet....</p>
        </article>
