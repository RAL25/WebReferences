Tag HTML <iframe>

Definição

A tag <iframe> especifica um quadro embutido. Um quadro embutido é usado para incorporar outro documento no documento HTML atual.

Dica: Use CSS para estilizar o <iframe> ( veja o exemplo abaixo ).

Dica: É uma boa prática sempre incluir um título atributo para o <iframe>. Isto é usado pelos leitores de tela para ler qual é o conteúdo do <iframe> é.

Atributos

Tag Valor Descrição
allow Especifica uma política de recursos para o <iframe>
allowfullscreen true
false		 Defina como verdadeiro se o <iframe> puder ativar o modo de tela cheia chamando o método requestFullscreen()
allowpaymentrequest true
false		 Definido como verdadeiro se um <iframe> de origem cruzada tiver permissão para invocar a API de solicitação de pagamento
height pixels Especifica a altura de um <iframe>. A altura padrão é 150 pixels
loading eager
lazy		 Especifica se um navegador deve carregar um iframe imediatamente ou adiar o carregamento de iframes até que algumas condições sejam atendidas
name text Especifica o nome de um <iframe>
referrerpolicy no-referrer
no-referrer-when-downgrade
origin
origin-when-cross-origin
same-origin
strict-origin-when-cross-origin
unsafe-url		 Especifica quais informações de referência serão enviadas ao buscar o iframe
sandbox allow-forms
allow-pointer-lock
allow-popups
allow-same-origin
allow-scripts
allow-top-navigation		 Habilita um conjunto extra de restrições para o conteúdo em um <iframe>
src URL Especifica o endereço do documento a ser incorporado no <iframe>
srcdoc HTML_code Especifica o conteúdo HTML da página a ser mostrada no <iframe>
width pixels Especifica a largura de um <iframe>. A largura padrão é 300 pixels

Atributos Globais

A tag <iframe> também suporta o Atributos globais em HTML.

Atributos Especificos

A tag <iframe> também suporta o Atributos de eventos em HTML.

Exemplo

Um quadro embutido é marcado da seguinte maneira:

        
        <iframe src="https://www.w3schools.com" title="W3Schools Free Online Web Tutorials"></iframe>
