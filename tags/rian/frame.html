Tag HTML <frame>

Não suportado em HTML5.

O <frame> tag foi usada no HTML 4 para definir uma janela específica ( frame ) dentro de um conjunto de quadros.

O que usar em vez disso?

Exemplo

Use o < iframe > tag para incorporar outro documento no documento HTML atual:

    <iframe src="https://www.ifnmg.edu.br"></iframe>
