Tag HTML <a>

Definição

A tag <a> define um hiperlink, que é usado para vincular de uma página a outra. O atributo mais importante do <a> elemento é o href atributo, que indica o destino do link.Por padrão, os links aparecerão da seguinte maneira em todos os navegadores:

  • Um link não visitado é sublinhado e azul
  • Um link visitado é sublinhado e roxo
  • Um link ativo é sublinhado e vermelho

    • Atributos

    Atributo Valor Descrição
    download filename Especifica que o destino será baixado quando um usuário clicar no hiperlink
    href URL Especifica o URL da página para onde o link vai
    hreflang language_code Especifica o idioma do documento vinculado
    media media_query Especifica para qual mídia/dispositivo o documento vinculado está otimizado.
    ping list_of_URLs Especifica uma lista de URLs separadas por espaços para as quais, quando o link for seguido, as solicitações de postagem com o corpo ping serão enviadas pelo navegador (em segundo plano). Normalmente usado para rastreamento.
    referrerpolicy no-referrer
    no-referrer-when-downgrade
    origin
    origin-when-cross-origin
    same-origin
    strict-origin-when-cross-origin
    unsafe-url    		 Especifica quais informações de referência serão enviadas com o link
    rel alternate
    author
    bookmark
    external
    help
    license
    nofollow
    noreferrer
    noopener
    prev
    search
    tag    		 Especifica a relação entre o documento atual e o documento vinculado
    target _blank
    _parent
    _self
    _top    		 Especifica onde abrir o documento vinculado
    type media_type Especifica o tipo de mídia do documento vinculado

    Atributos Globais

    A tag <a> também suporta o Atributos globais em HTML.

    Atributos Especificos

    A tag <a> também suporta o Atributos de eventos em HTML.

    Exemplo

    Como abrir um link em uma nova janela do navegador:

            
        <a href="https://www.ifnmg.edu.br" target="_blank">Visite o site do IFNMG</a>
