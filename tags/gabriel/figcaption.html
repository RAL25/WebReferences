Tag HTML <figcaption>

Definição

A tag <figcaption> define uma legenda para um elemento <figure>. O elemento <figcaption> pode ser colocado como o primeiro ou último filho do elemento <figure>.

Atributos

Atributo Valor Descrição
- - -

Atributos Globais

A tag <figcaption> suporta atributos globais do HTML.

Atributos de Eventos

A tag <figcaption> suporta atributos de enventos do HTML.

Exemplo

Use um elemento <figure> para marcar uma foto em um documento e um elemento figcaption para definir uma legenda para a foto:

        
        <figure>
            <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
            <figcaption> Fig.1 - Trulli, Puglia, Italy. </figcaption>
        </figure>
